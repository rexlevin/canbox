## 1. 日志查看器

### 基本信息
- **完成日期**: 2026-03-14
- **优先级**: ⭐⭐⭐⭐⭐ 高
- **难度**: 🟡 中等
- **相关文件**: `modules/main/logManager.js`, `src/components/LogViewer.vue`

### 功能描述
提供独立的日志查看窗口，支持实时日志查看、过滤、搜索和导出等功能，提升开发和问题排查效率。

### 核心特性

#### 日志显示
- 实时日志显示，支持自动滚动到最新日志
- 限制显示数量（最多 500 条），避免性能问题
- 智能滚动：用户查看历史日志时不自动滚动，滚动到底部时恢复自动滚动

#### 日志过滤
- 按日志级别过滤（debug/info/warn/error）
- 支持多选，可同时查看多个级别的日志
- "显示全部"快速重置过滤器

#### 搜索功能
- 关键词搜索，支持普通文本匹配和正则表达式
- 搜索结果高亮显示
- 清空搜索恢复原始显示

#### 日志导出
- 导出为 .txt 格式（纯文本）
- 导出为 .json 格式（结构化数据）
- 支持导出全部日志或最近 N 条日志

#### 日期导航
- 日期选择器，按日期查看历史日志
- 自动列出所有可用的日志文件（按日期）
- 支持加载和解压历史日志文件（.gz 压缩格式）

#### 日志管理
- 清空日志文件：清空当日日志内容，不删除文件
- 清理历史日志：显示预览对话框，选择删除特定日期的日志文件
- 日志保留天数配置：1-365 天可调整

#### 多日志源支持
- 应用日志（app.log）
- 监控日志（monitor.log）
- 切换日志源时自动更新可用日期列表

#### 窗口特性
- 独立窗口，可在任务栏单独显示
- 可置顶选项，保持窗口在最前
- 保存和恢复窗口状态（位置、大小、置顶状态）
- 通过系统托盘菜单快速打开

### 技术要点

#### 后端实现
- 使用 log4js 的 dateFile appender 按日期分割日志
- 实现日志 ID 生成，支持增量获取日志
- 统一的日志目录路径管理（`getLogDir()` 函数）
- IPC 处理器：`get-logs`, `get-log-files`, `export-logs`, `clear-logs`, `cleanup-old-logs`
- 支持中英文国际化

#### 前端实现
- Vue 3 Composition API + Element Plus 组件库
- 轮询机制获取增量日志（1 秒间隔）
- 虚拟滚动优化性能（最多显示 500 条）
- 使用 `v-for` 的 `:key` 绑定唯一 ID 避免重复渲染
- 支持正则表达式搜索和关键词高亮

### 使用示例

```javascript
// 打开日志查看器
canbox.log.openViewer();

// 获取最近日志
canbox.log.getRecent({ limit: 100 })
  .then(logs => {
    console.info('Recent logs:', logs);
  });
```

## 2. 自定义字体设置

### 基本信息
- **完成日期**: 2026-03-21
- **优先级**: ⭐⭐⭐ 中
- **难度**: 🟢 简单
- **相关文件**: `ipcHandlers.js`, `src/components/Settings.vue`

### 功能描述
为 Canbox 添加全局字体选择功能，用户可以在"设置"页面中选择自己喜欢的字体，该设置会立即应用到整个应用。

### 实现方式

#### 后端实现（Electron 主进程）
**文件**: `ipcHandlers.js`

添加了两个 IPC 处理器：
```javascript
// 获取字体设置
ipcMain.handle('font-get', () => {
  const canboxStore = getCanboxStore();
  const savedFont = canboxStore.get('font', 'default');
  return savedFont;
});

// 设置字体
ipcMain.handle('font-set', async (event, fontValue) => {
  const canboxStore = getCanboxStore();
  canboxStore.set('font', fontValue);

  // 通知所有窗口字体已更改
  BrowserWindow.getAllWindows().forEach(win => {
    win.webContents.send('font-changed', fontValue);
  });
});
```

#### 前端实现（Vue 组件）
**文件**: `src/components/Settings.vue`

添加字体选择组件：
```vue
<template>
  <el-select v-model="selectedFont" @change="onFontChange">
    <el-option label="系统默认" value="default" />
    <el-option label="微软雅黑" value='"Microsoft YaHei", sans-serif' />
    <el-option label="思源黑体" value='"Source Han Sans SC", sans-serif' />
    <el-option label="苹方" value='"PingFang SC", sans-serif' />
    <el-option label="自定义..." value="custom" />
  </el-select>
</template>
```

### 存储格式
```json
{
  "font": "\"Microsoft YaHei\", sans-serif"
}
```

## 3. 用户数据路径管理

### 基本信息
- **完成日期**: 2026-02-11
- **优先级**: ⭐⭐⭐⭐ 高
- **难度**: 🟡 中等
- **相关文件**: `modules/main/storageManager.js`

### 功能描述
支持指定自定义的用户数据存储路径，允许用户将数据存储在特定位置而非默认位置。

### 核心特性
- 运行时指定数据存储路径
- 支持相对路径和绝对路径
- 路径验证和权限检查
- 数据迁移工具

### 使用示例
```javascript
// 设置自定义数据路径
canbox.config.setDataPath('/custom/data/path');

// 获取当前数据路径
const path = canbox.config.getDataPath();
```
