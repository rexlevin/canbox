# 工具目录

本目录包含Canbox项目的辅助工具。

## 当前状态

**注意**：本目录目前为空，因为根据项目决策，我们采用纯指导方案而非自动化工具。

## 文档同步流程

当OpenSpec有变更时，AI Agent应按照以下文档进行工作：

### 核心指导文档
- **`docs/AI_AGENT_SYNC_GUIDE.md`** - 完整的同步操作指南

### 主要步骤
1. **阅读指南**：仔细阅读 `AI_AGENT_SYNC_GUIDE.md`
2. **提取信息**：从OpenSpec变更目录读取关键信息
3. **创建文档**：参考 `docs/changes/template.md` 创建人类友好文档
4. **更新汇总**：编辑 `docs/changes/completed/README.md`

### 参考文档
- `docs/changes/template.md` - 变更文档模板
- `docs/changes/completed/README.md` - 已完成变更汇总
- `docs/changes/completed/` 下的现有文件 - 参考示例

## 为什么没有自动化工具？

经过讨论，我们决定采用方案B+D组合：
- **方案B**：详细的指导文档
- **方案D**：纯交互式指导

这种组合的**优点**：
1. **极简**：不需要维护复杂的工具
2. **灵活**：AI Agent可以根据实际情况调整
3. **可控**：人类可以随时介入和审核
4. **稳定**：不会因为工具bug导致问题

## 未来考虑

如果未来需要自动化工具，可以考虑：
1. 基于现有指导文档开发简单脚本
2. 只在确实需要时添加
3. 保持工具的简单性和可维护性
